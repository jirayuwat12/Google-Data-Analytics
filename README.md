# Google-Data-Analytics Professional Certificate
[from ***Google Data Analytics Professional Certificate*** on Coursera](https://www.coursera.org/professional-certificates/google-data-analytics)

## About this Professional Certificate
This Professional Certificate program is designed to help you gain the skills you need to become a data analyst. You will learn to organize data, uncover patterns and insights, draw meaningful conclusions, and clearly communicate critical findings. In the final Capstone Project, you will apply your skills to a real-world task by analyzing data to create a data-driven solution. This program will help you gain the skills you need to become a data analyst.
